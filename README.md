# 📋 Lista de Tarefas - React

Projeto desenvolvido com o objetivo de praticar os principais conceitos de **React** e **JavaScript**, utilizando componentes, gerenciamento de estado, manipulação de eventos e persistência de dados no navegador.

## 🚀 Objetivo

Este projeto foi criado como parte dos meus estudos em React, buscando aplicar na prática conceitos fundamentais do desenvolvimento Front-end.

Durante o desenvolvimento foram explorados tópicos como:

- Componentes de classe (Class Components)
- Componentização da aplicação
- Props
- State
- Manipulação de eventos
- Ciclo de vida dos componentes
- Renderização dinâmica com `map()`
- Atualização de listas
- Edição e remoção de itens
- Persistência de dados utilizando `localStorage`
- Validação de entradas
- Organização de componentes e arquivos CSS

---

## 📸 Funcionalidades

- ✅ Adicionar tarefas
- ✏️ Editar tarefas
- 🗑️ Remover tarefas
- 💾 Salvar automaticamente as tarefas no navegador
- 🔄 Recuperar as tarefas após recarregar a página
